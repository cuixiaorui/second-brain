## 部署步骤
使用腾讯云的 cli 工具 cloudbase

- 在需要上传的文件夹执行  
```js
// -e 后面的是 坏境 id
// 可以在腾讯云云开发获取到
cloudbase hosting:deploy -e element3-0gsg8qiu30d7313a
```
它会把当前文件夹下面的所有内容上传到腾讯云的静态网站托管上
