
当英语当中的句子拆到不能再拆的时候，都是在说”什么“+ ”怎么样“

- 什么 = 主语
- 怎么样 = 谓语


## 五种基本句型

1. 无承受者 - 主语 + 动词
	- 我累了
2. 一个承受者 - 主语 + 动词 + 宾语
	- 我喜欢你
3. 两个承受者 - 主语 + 动词 + 间接宾语 + 直接宾语
	- 我教你英语
4. 只有一个动作承受者(但是需要补充) - 主语 + 动词 + 宾语 + 补语
	- 我认为你很瘦
	- 只有宾语无法描述清楚，必须加补充语句
5. 把这个词后面的信息赋予给前者 - 主语 + 动词（系动词） +  补语
	- you are cute
	- 你 = 可爱
